Plain-language summary

Every problem is self-contained: the data are simulated in the same code block as the model. Run the code blocks in order, then open the <details> to see the full worked solution and biological interpretation.

Forest carnivores are perfect Bayesian problems: rare, hard to detect, hierarchically grouped (by basin, by year, by camera array), and managed for population recovery where derived quantities (occupancy, density, survival) matter more than raw counts.

Problem 1 · 20 min · hierarchical Poisson

Marten detection counts on baited camera arrays

Twelve camera arrays across the Oregon coast range each ran for 30 nights. Total marten detections at array $j$ are modelled as:

$y_j \sim \text{Poisson}(E_j \cdot \lambda_j), \quad \log \lambda_j \sim \mathcal{N}(\mu, \sigma_\lambda^2)$

where $E_j = 30$ camera-nights is offset effort. Fit in NIMBLE.

Problem 7 · 25 min · prior sensitivity

Marten detection: does the prior matter?

Re-fit Problem 1 with three priors on $\mu$:

  1. Vague: $\mu \sim \mathcal{N}(0, \sigma^2 = 10000)$
  2. Weakly informative: $\mu \sim \mathcal{N}(0, \sigma^2 = 4)$
  3. Informative (centered on prior belief from previous studies): $\mu \sim \mathcal{N}(-2, \sigma^2 = 0.25)$

Compare the posterior of $\mu$ for each.
