## How to add a description

1. Open the lecture-notes HTML file and find the figure note (search for `needs description`).
2. Replace the sentence "This slide contains an image... Instructor: replace this sentence..." with 1–3 sentences describing what the figure shows and its takeaway.
3. For equations shown as images, you can instead write the equation in LaTeX between `$...$` so it renders as screen-reader-navigable math.
