Day 1 · Alternate Problem Set v2

Marine and fisheries themed problems

Novel problems covering the same concepts as the main lab — sample-space reasoning, Bayes, Bernoulli/Binomial/Poisson/NB/Multinomial — but in a marine and fisheries context.

This problem set is designed to take roughly the same time as the main lab. Answers (with R verification) appear at the end. Try every problem first, then check.

Problem 1 · conditional probability

Acoustic detection of tagged steelhead

A tagged steelhead passes through a river reach with two acoustic receivers. Each receiver, independently and given the fish is present, detects the tag with probability 0.85. If the fish is absent, false detections are negligible.

  1. Given a fish is present, what is the probability that at least one receiver detects it?
  2. Given at least one receiver detects the fish, what is the probability both detected it?
  3. Why is your answer to (b) different from $0.85$?
Problem 2 · Bayes' rule

eDNA detection of invasive carp

In a lake, the prior probability that bighead carp are present is 0.20. An eDNA assay has sensitivity 0.92 (P(+|present)) and specificity 0.97 (P(−|absent)).

  1. If a sample is positive, what is the posterior probability the carp are present?
  2. If two independent samples both come back positive, what is the posterior?
  3. How much does the answer change if the prior probability is 0.02 instead of 0.20?
Problem 3 · Binomial

Sea turtle nest success

A monitored loggerhead beach has $n=18$ nests, each with hatching success probability $p = 0.62$ (assume independence).

  1. Expected number of successful nests? Variance?
  2. P(exactly 12 successful nests)?
  3. P(at least 15 successful nests)?
  4. If you wanted at least an 80% chance of seeing $\ge 15$ successes, what would $p$ need to be?
Problem 4 · Bernoulli + indicators

Sea otter neighborhood survival

$n = 20$ tagged sea otters in a kelp forest each survive a year with probability 0.9 independently. Count the number of "isolated mortality" events — an otter that died, whose immediate two neighbors (by tag ID order) both lived.

  1. Use the indicator method to compute the expected number of isolated mortality events. Treat tag IDs as a linear (non-circular) arrangement; consider only the 18 interior otters.
  2. Verify by simulation (20,000 reps).
Problem 5 · Poisson

Albatross bycatch in a longline fishery

Bycatch events of black-footed albatross occur at rate $\lambda = 0.4$ per 1000 hooks set.

  1. If a vessel sets 5000 hooks on one trip, what distribution describes the bycatch count? Give mean and variance.
  2. P(zero bycatch)?
  3. P(more than 3 albatross taken)?
  4. Of 25 independent trips, what is the expected number of trips with zero bycatch?
Problem 6 · Multinomial

Bycatch species composition

In a shrimp trawl in the Gulf of Mexico, the species composition of bycatch is shrimp 0.55, croaker 0.20, drum 0.15, other 0.10. A single tow yields 200 fish.

  1. What is the joint probability of seeing exactly (110, 40, 30, 20)?
  2. Simulate 1000 tows and look at the marginal distribution of "drum."
  3. What is the name of the marginal distribution?
Problem 7 · Negative binomial

Designing a trawl survey for over-dispersed counts

Pre-survey power analysis: catches per tow have historical mean = 25 fish and variance = 200 fish. You're choosing between fitting a Poisson and a NB model.

  1. What NB size parameter $k$ matches the historical variance?
  2. Simulate 500 tows under both models. Compare the empirical P(0) and P(catch > 60).
  3. If your science question is "how often will we get a zero catch", which model gives you a better answer?
Problem 8 · Binomial inference

Tag-recapture

You release 100 marked salmon smolts and recapture 8 of them out of a sample of 50. (Hypergeometric approximated by binomial when population is large.)

  1. Write the binomial data-generating model for the recapture count under the assumption that the recapture probability is $p$.
  2. If the true $p = 0.10$, what is the probability of recapturing at least 8 out of 50?
  3. What does this say about the value of a single mark-recapture run as evidence for any specific $p$?
Problem 9 · Poisson + log link

Microplastic particles in mussels

Counts of microplastic particles per mussel follow Poisson($\lambda$). $\lambda$ varies across sites. At site A, $\lambda = 2$; at site B, $\lambda = 8$.

  1. If you pool 50 mussels from site A and 50 from site B and treat them as a single Poisson sample, what is the empirical mean and variance? Will the pooled distribution still be Poisson? Why or why not?
  2. Show by simulation that the pooled distribution is overdispersed.
  3. What does this exercise tell you about why real count data are over-dispersed, even when the underlying process is Poisson at each site?
Problem 10 · expected value

Voter on the dock

20 commercial fishers gather to vote on a regulation. Each will vote "yes" independently with probability 0.4. The motion passes if more than half vote yes.

  1. What is the expected number of yes votes? Variance?
  2. What is the probability the motion passes?
  3. What is the smallest number of voters $n$ for which the probability of passage (with $p=0.4$) drops below 1%?

Answers (with R)

Problem 1

(a) $1 - 0.15^2 = 0.9775$.

(b) $P(\text{both} \mid \ge 1) = 0.85^2 / 0.9775 = 0.7392$.

(c) Knowing "at least one detected" raises both joint outcomes proportionally — but the (lone) one-detector outcomes still get probability mass.

Problem 2
p_pres <- 0.20
sens <- 0.92; fpr <- 0.03
# (a) one positive
(sens*p_pres) / (sens*p_pres + fpr*(1-p_pres)) # 0.8846
# (b) two positives — update twice
post1 <- (sens*p_pres) / (sens*p_pres + fpr*(1-p_pres))
(sens*post1) / (sens*post1 + fpr*(1-post1)) # 0.9959
# (c) prior = 0.02
p_pres <- 0.02
(sens*p_pres) / (sens*p_pres + fpr*(1-p_pres)) # 0.3848

With prior = 0.02, even after one positive the posterior is only ~38%. Base rate dominates.

Problem 3
n <- 18; p <- 0.62
n*p; n*p*(1-p) # mean 11.16, var 4.24
dbinom(12, n, p) # 0.181
1 - pbinom(14, n, p) # 0.0466
# find p s.t. P(X>=15) >= 0.8
ps <- seq(0.5, 1, 0.005)
ps[which(1 - pbinom(14, n, ps) >= 0.8)[1]] # ~ 0.87
Problem 4

(a) For an interior otter $i$ (positions 2..19), $P(X_i = 1) = (1-0.9) \cdot 0.9 \cdot 0.9 = 0.081$. Expected count = $18 \cdot 0.081 = 1.458$.

set.seed(1); reps <- 2e4
counts <- replicate(reps, {
 y <- rbinom(20, 1, 0.9)
 d <- !y
 sum(d[2:19] & y[1:18] & y[3:20])
})
mean(counts) # ~ 1.46

Problem 7

Variance = $\mu + \mu^2/k$ → $200 = 25 + 625/k$ → $k = 625/175 = 3.57$.

set.seed(1); n <- 500
pois <- rpois(n, 25)
nb <- rnbinom(n, mu=25, size=3.57)
mean(pois==0); mean(nb==0) # ~ 0 vs ~ 0.04
mean(pois>60); mean(nb>60) # ~ 0 vs ~ 0.07

(c) Use NB — Poisson assigns essentially zero probability to outcomes that the historical data tell you happen routinely.

Problem 8

(a) $R \sim \text{Bin}(50, p)$.

1 - pbinom(7, 50, 0.10) # 0.260

(c) Lots of "consistent with $p=0.10$" results, but also consistent with $p=0.16$, $p=0.20$, etc. A single recapture run is informative but doesn't pin down $p$ tightly — a clean lead-in to confidence intervals and likelihood.

Problem 9
set.seed(1)
a <- rpois(50, 2); b <- rpois(50, 8)
pooled <- c(a, b)
mean(pooled); var(pooled) # ~ 5 and ~ 12
# Pooled is NOT Poisson; the "mixed Poisson" has variance > mean.

(c) This is exactly why count data are over-dispersed in practice — heterogeneity across units (sites, individuals, conditions). It is the mechanism behind the Negative Binomial: a Poisson with a gamma-distributed rate.

Problem 10
n <- 20; p <- 0.4
n*p; n*p*(1-p) # mean 8, var 4.8
1 - pbinom(10, n, p) # P(pass) = P(>=11) = 0.128
# n* : smallest n where P(>n/2) < 0.01
ns <- 20:200
idx <- min(which(sapply(ns, function(n) 1 - pbinom(n/2, n, 0.4)) < 0.01))
ns[idx] # ~ 96
