Problem 1 · pooling decision

Cougar home-range size across mountain ranges

A wildlife biologist measures the 95% MCP home-range size (km$^2$) of 60 GPS-collared cougars, distributed across 8 mountain ranges (5–10 cougars per range). The biological question: is there a real difference in mean home-range size among mountain ranges?

  1. Write out three candidate models — complete pooling, no pooling (range as fixed effect), partial pooling (range as random effect) — in mathematical notation, including all coefficients and distributions.
  2. Which model would you fit if you wanted to make inferences about cougars in other mountain ranges (not in this dataset)? Why?
  3. For 4 of the ranges, only 5 cougars were collared; for the other 4 ranges, 10 cougars were collared. Will partial pooling shrink the 5-cougar ranges more or less than the 10-cougar ranges, and why?

Problem 6 · LRT vs AICc

m2 <- lmer(Attend ~ year + spring_precip + (1|Lek), REML=FALSE, data=g)
m3 <- lmer(Attend ~ year * spring_precip + (1|Lek), REML=FALSE, data=g)

anova(m0, m1, m2, m3)

Problem 8 · BIC vs AIC

Coyote pup survival: when BIC misses the weak effect

You want to know whether 3 predictors affect coyote pup survival: litter_size (strong known effect), maternal_age (modest effect), and winter_severity (weak effect). $n = 60$ pups. Simulate:

set.seed(7); n <- 60
litter <- runif(n); age <- runif(n); winter <- runif(n)
y <- 0.8*litter - 0.4*age + 0.1*winter + rnorm(n, 0, 0.5)
d <- data.frame(y, litter, age, winter)
m <- lm(y ~ litter + age + winter, data=d)
dredge(m, rank="AIC"); dredge(m, rank="AICc"); dredge(m, rank="BIC")
  1. Run the simulation. Which predictors does each criterion retain in the top model?
  2. Now wrap it in a 500-rep loop and report the proportion of times each criterion retains winter.
  3. Which criterion would you use if your science question is "is winter severity worth studying further?", and which if it's "does the winter signal generalize out of sample?"
